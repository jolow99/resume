# Resume

A modern resume built with Typst.

## Files

- `main.typ` - Resume source file in Typst format
- `main.pdf` - Generated PDF resume

## Building

To compile the resume, you'll need [Typst](https://typst.app/) installed:

```bash
typst compile main.typ
```

## Template Credit

This resume is based on the [chicv](https://github.com/skyzh/chicv) template.
